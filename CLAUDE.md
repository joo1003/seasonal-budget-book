# 개발 환경 안내

이 폴더는 **개발 연습용** 작업공간입니다.

## 기술 스택

- **Next.js** — 웹사이트를 만드는 프레임워크
- **Supabase** — 데이터베이스와 로그인 기능을 제공하는 서비스
- **Vercel** — 만든 웹사이트를 인터넷에 올리는(배포) 서비스

## Claude 행동 지침

### 설명 방식
- 개발 용어는 비개발자도 이해할 수 있게 쉬운 말로 설명한다
- 지금 무엇을 하고 있는지 자주 알려준다
- 예: "지금 로그인 화면을 만들고 있어요", "데이터베이스에 테이블을 추가하고 있어요"

### 작업 방식
- 사용자가 지시한 것만 단계적으로 수행한다
- 요청하지 않은 기능 추가, 리팩토링, 미래 대비 코드는 작성하지 않는다
- 할 수 있는 것은 최대한 직접 처리한다
- 사람이 꼭 직접 해야 하는 것(예: Vercel 배포 버튼 클릭, 환경변수 입력)만 쉬운 말로 안내한다

### 작업 완료 후
- 개발이 끝나면 항상 바로 확인할 수 있도록 브라우저 미리보기를 열어준다
- 배포가 끝나면 접속 가능한 주소를 알려준다
